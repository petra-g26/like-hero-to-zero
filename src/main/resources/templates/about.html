Über das Projekt

Like Hero To Zero ist eine wissenschaftliche Webanwendung zur Verwaltung und Analyse von CO₂-Emissionsdaten verschiedener Länder.

Ziel des Projekts ist die übersichtliche Darstellung, Erfassung und Analyse von Emissionswerten in einer modernen, webbasierten Plattform.

Verwendete Technologien

  • Java
  • Spring Boot
  • Thymeleaf
  • Bootstrap 5
  • GitHub
